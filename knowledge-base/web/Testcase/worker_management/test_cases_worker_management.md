## Tóm tắt phạm vi

**Trong scope:**
- Functional: List view + Search + Filter + Pagination + Profile Details + Pass + Delete + Open Inception
- Document view (read-only): Right to Work (Passport, ID Card, Birth Certificate, Share code, NI document) — chỉ xem dữ liệu có sẵn, KHÔNG có chức năng upload/replace/delete
- Document upload: Skill Card (CSCS, CPCS, CISRS, NPORS, Other, Not applicable)
- Multi-locale
- UI states (loading, empty, error, responsive 1280px+)
- Security: XSS, IDOR, file upload abuse (Skill Card only), authz bypass, signed URL verification (RtW documents)

**Ngoài scope:**
- Integration với hệ thống ngoài (HR, AD)
- Performance/load test với workforce lớn
- Full WCAG accessibility
- Email notification flow
- Permission matrix giữa các role (stgeo full permission per Q16)

**Compliance/Security findings ghi nhận (cần escalate cho PO/Legal/Security):**
- **F-COMP-01:** Hard delete (Q18) cho hệ thống compliance UK — vi phạm potential GDPR retention. **Escalate Legal/DPO**
- **F-COMP-02:** Không audit trail cho Worker status transition (Q17) — status được set bởi backend/system nhưng không có log ai/khi nào/evidence; vi phạm potential traceability requirement compliance UK
- **F-SEC-01:** Không bắt buộc format/size file upload (Q14) — áp dụng cho **M5 Skill Card** (M4 RtW chỉ view, không upload). Risk: arbitrary upload, XSS qua SVG, DOS qua file lớn
- **F-SEC-03:** Document RtW URL phải là signed URL có expiration — nếu là public URL → risk data leak (PII document như Passport/ID Card có thể bị truy cập không cần auth)
- **F-SEC-02:** Không rate limit Delete (Q24) — race condition, brute-force
